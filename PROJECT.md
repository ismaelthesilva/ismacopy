## Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS
- **Backend:** Node.js, TypeScript, Next.js API routes
- **Database:** PostgreSQL (Neon)
- **ORM:** Prisma
- **Cloud:** AWS (Lambda, S3)
- **Authentication:** Supabase

## Studio System & App Structure

- `apps/web/` — Next.js product platform
  - `components/` — Modular UI & system logic
  - `contexts/` — Global state (Theme, Language)
  - `lib/` — Core utilities (auth, data, integrations)
  - `locales/` — i18n for global reach
  - `public/` — Assets (images, data)
  - `app/` — App router, pages, and flows

## How to Run Locally

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Configure environment variables (see `.env.example`)
4. Start the development server:
   ```sh
   npm run dev
   ```
